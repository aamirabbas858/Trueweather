# True Weather 🌤️

My first weather app built as a 2nd-year CS student. Enter any city and get live weather data with icons!

## Live Demo
[![True Weather](https://img.shields.io/badge/Live-True%20Weather-brightgreen)](https://abbasaamir.github.io/trueweather)


## Features
- ✅ Live weather for any city worldwide
- 🌡️ Temperature + "feels like"
- 🌈 Official OpenWeather icons  
- 💧 Humidity percentage
- 📱 Works on mobile + desktop
- ⌨️ Press Enter to search
- 🛡️ Error handling + loading states
